How to Test

  1. Select Text: Highlight any prompt above with your mouse
  2. Look for Tooltip: A tooltip should appear offering to optimize the text
  3. Click Tooltip: Click the tooltip to open the extension popup
  4. Or Use Extension Icon: Click the UsersPrompts icon in your Chrome toolbar
  5. Enter Prompt: Paste or type a prompt in the form
  6. Optimize: Click "Optimize Prompt" to see the results

Expected Results

After optimization, you should see:

  • Improved Prompt: A comprehensive, well-structured system prompt
  • Detailed Scoring: Scores across clarity, specificity, completeness, effectiveness, and robustness
  • What Was Wrong: Analysis of issues with the original prompt
  • What Was Improved: Specific improvements made to the prompt
  • Recommendations: Advice for further improvements
  • Improvement Metrics: Overall improvement percentage and points gained

Troubleshooting

Extension not working?

  • Make sure the extension is loaded in Chrome
  • Check that your backend server is running on localhost:8000
  • Open Chrome DevTools to see any error messages

API connection issues?

  • Verify the backend API is accessible
  • Check CORS settings on your backend
  • Test API endpoints directly with curl or Postman
